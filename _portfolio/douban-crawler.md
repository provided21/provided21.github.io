A Python-based web crawler designed to automatically collect and structure movie metadata from Douban (豆瓣), China's leading film and book review platform. Built as part of the AI Programming coursework.

**Key Features:**
- Automated data extraction from Douban movie pages
- Structured output in clean tabular format
- Anti-scraping countermeasures implemented

**Tech Stack:** Python, web scraping, data parsing
