An AI assistant application that integrates with the Model Context Protocol (MCP), enabling dynamic tool use and extensible agent capabilities. Built as part of the AI Programming coursework at NJU.

**Key Features:**
- MCP protocol integration for standardized tool calling
- Extensible architecture supporting custom tools and plugins
- Open-source release on GitHub

**Tech Stack:** Python, MCP protocol, LLM APIs
